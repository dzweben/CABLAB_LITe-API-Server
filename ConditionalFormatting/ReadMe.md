# 🎨 Conditional Formatting Automation for Follow-Up Sheets

This script uses the **Google Sheets API** to apply conditional formatting rules across multiple columns in the `Follow up.1` tab of our participant tracking spreadsheet. It automatically highlights cells based on completion data in the `sts1w1` tab.

---

## 💡 Purpose

The formatting rules help visually flag survey completions for two different waves (e.g., Wave 1 and Follow-up), using values generated by the REDCap-to-Google Sheets scripts in the `FollowUpCheck/` folder.

This enables real-time, color-coded tracking of survey completion status across participant waves — keeping things clean, visible, and up-to-date without manual effort.

---

## 🔁 What This Script Does

- Authenticates with the Google Sheets API using a service account
- Matches participant IDs between `Follow up.1` and `sts1w1` tabs
- Applies a **custom formula rule** using `INDEX`, `MATCH`, and `INDIRECT` to compare completion codes
- Highlights cells in specified columns in `Follow up.1` **if the matched cell in `sts1w1` has a value of `2` (complete)**
- Applies a green background color for completed entries

---
